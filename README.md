# personal-introduction-program
## 📌 Project Overview
This program takes user input like name, age, and hobby and displays a friendly message.

## ⚙️ Setup Instructions
1. Install Python
2. Open IDLE
3. Run the program

## 🧩 Code Structure
- Uses input()
- Uses variables
- Uses print()
- Uses f-strings

## 🛠️ Technical Details
Concepts used:
- Variables
- Input/Output
- String formatting

## ✅ Testing
Example:
Input:Mariya,22,Baking
Output: Hello Mariya! You are 22 years old and you enjoy Baking.
